Cheat Sheet for HTML and CSS

HTML

Tags Name Description
TABLES
<table> Table Defines a table.
<thead> Table Head Groups the header content in a table.
<tbody> Table Body Groups the body content in a table.
<tr> Table Row Defines a row in a table.
<th> Table Header Defines a header cell in a table.
<td> Table Data Defines a cell in a table.
TEXT CONTENT
<h1> to <h6> Header 1 to Header 6 Defines HTML headings.
<p> Paragraph Defines a paragraph.
<br> Line Break Inserts a line break.
<hr> Horizontal Rule Defines a thematic change in the content.
<span> Content Span Defines a section in a document.
FORMS AND LISTS
<form> Form Defines an HTML form for user input.
<input> Input Defines an input control.
<ul> Unordered Lists Defines an unordered list.
<ol> Ordered Lists Defines an ordered list.
<li> Lists Defines a list item.
<nav> Navigation Defines a set of navigation links.
<main> Main Specifies the main content of a document.
<footer> Footer Defines a footer for a document or section.
<div> Division Defines a division or section.
MULTIMEDIA AND EMBEDDED CONTENT
<audio> Audio Embeds sound content.
<video> Video Embeds video content.
<source> Source Specifies multiple media resources for <audio> and <video>
<embed> Embed Embeds external content (like a plugin).
<iframe> Inline Frame Embeds external content (like a plugin).
ADVANCE HTML
<article> Article Represents a self-contained piece of content.
<section> Section Defines a section in a document.
<aside> Aside Defines content aside from the main content (like a sidebar).
<figure> Figure Specifies self-contained content, such as illustrations, diagrams, or photos.
<figcaption> Figure Caption Defines a caption for a &<figure> element.

CSS

Name Description
TEXT STYLING
color Sets the color of text.
font-size Sets the size of the font.
font-family Specifies the font for text.
font-weight Sets the weight (boldness) of the font.
text-align Aligns text horizontally.
TEXT DECORATION AND LAYOUT
text-decoration Adds decoration to text (underline, overline, etc.).
line-height Sets the height of lines of text.
margin Sets the margin area around elements.
padding Sets the padding area within elements.
border Sets the border around elements.
ADVANCED SIZE AND DIMENSIONS
flex A shorthand property for the flex-grow, flex-shrink, and flex-basis properties.
flex-grow Defines the ability for a flex item to grow if necessary.
flex-shrink Defines the ability for a flex item to shrink if necessary.
flex-basis Sets the initial size of a flex item.
aspect-ratio Sets the preferred aspect ratio for an element.
object-fit Specifies how the content of a replaced element should be resized to fit its container.
object-position Specifies the alignment of the replaced element inside its container.
ADVANCED POSITIONING
position Specifies the type of positioning method used for an element (static, relative, absolute, fixed, sticky).
z-index Specifies the stack order of an element.
transform Applies a 2D or 3D transformation to an element (e.g., translate, rotate, scale).
perspective Gives an element a 3D-positioned perspective view.
perspective-origin Defines the origin for the perspective property.
backface-visibility Defines whether or not the back face of an element is visible when facing the user.
clip-path Defines a clipping region for an element.
grid-template-columns Defines the columns of a grid container.
LAYOUT AND EFFECTS
float Specifies whether an element should float to the left, right, or none.
clear Specifies on which sides of an element floating elements are not allowed.
overflow Specifies how to handle content that overflows an element's box.
opacity Sets the opacity level of an element.
cursor Specifies the type of cursor to be displayed when pointing over an element.
box-shadow Adds shadow effects to an element.
